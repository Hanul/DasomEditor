# Dasom Editor
JavaScript로 만들어진 소스코드 에디터입니다.

***Electron 4.2.0에서 사용 가능합니다.***

## 기능들
- FTP 기능
- Git 기능
- 파일 저장 시 명령 기능
- 폴더 내 모든 파일 검색 기능
- 외부 파일을 드래그 앤 드롭으로 복사
